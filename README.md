# Picard Geocache

This is a small Gatsby site to share a Star Trek themed puzzle that eventually leads to a geocache.

Check out the [deployed site](https://sfritton.github.io/picard-geocache/).

## Develop

```sh
npm run install
npm run develop
```

## Deploy

```sh
npm run deploy
```

## Dependencies

- [Gatsby](https://www.gatsbyjs.com/)
